# AI-Based_Resume_Matcher
AI Resume Job Matcher (NLP | Python | Gradio)
Built and deployed an AI-based resume screening application on Hugging Face Spaces
Implemented Natural Language Processing techniques using TF-IDF and Cosine Similarity to match resumes with job roles
Enabled resume upload (PDF/TXT) and extracted text for analysis
Generated top job recommendations with match scores based on skill similarity
Designed an interactive UI using Gradio for real-time user interaction
Live App: https://huggingface.co/spaces/Musarratsk10/resume-matcher-app
